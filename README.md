# Spotify Clone 🎵

A responsive Spotify UI clone built with pure HTML, CSS, and JavaScript.

## Preview

A pixel-accurate recreation of the Spotify web interface featuring a sidebar, scrollable main content, and a fixed music player bar.

## Features

- 🎨 Dark theme with Spotify's signature green accent (`#1db954`)
- 📱 Responsive card grid layout
- 🎵 Fixed music player bar with progress slider
- ❤️ Like/unlike toggle on the now-playing track
- ▶️ Hover-reveal play buttons on cards
- ✨ Staggered fade-up card animations on load
- 🔊 Volume slider control
- 🏠 Active state navigation highlighting

## Tech Stack

- HTML5
- CSS3 (CSS Grid, Flexbox, CSS Variables, Animations)
- Vanilla JavaScript
- [Font Awesome 6](https://fontawesome.com/) for icons
- [Figtree](https://fonts.google.com/specimen/Figtree) via Google Fonts

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR-USERNAME/spotify-clone.git
   ```

2. Open `index.html` in your browser — no build step or server required.

## Live Demo

Deploy instantly with [GitHub Pages](https://pages.github.com/):  
Go to **Settings → Pages → Source → main branch → / (root)** and save.
